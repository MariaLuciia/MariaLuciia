### 🚦 Projeto em Destaque: Denúncia Cidadã (Detran)
Desenvolvi uma solução digital para facilitar o registro de infrações de trânsito pelos cidadãos. 

* **O Problema:** Dificuldade e demora no reporte de infrações cotidianas.
* **A Solução:** Um Web App otimizado para telemóveis que permite anexar fotos e dados da placa instantaneamente.
* **Tecnologias:** HTML5, CSS3 e JavaScript.
* **Status:** Protótipo funcional (MVP) concluído.

🚀 [Detran - Denúncia Cidadã](https://marialuciia.github.io/detrans-cidadao/)

> *Nota: O código-fonte deste projeto é mantido em repositório privado por questões de propriedade intelectual.*
